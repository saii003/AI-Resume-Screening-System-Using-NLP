# AI Resume Screening System Using NLP

## 📌 Project Overview

This project is an AI-powered Resume Screening System that matches resumes with a given job description using Natural Language Processing (NLP).

The system converts resumes and the job description into TF-IDF vectors and calculates Cosine Similarity to rank resumes based on their relevance.

---

## 🚀 Features

- Resume Screening using NLP
- TF-IDF Vectorization
- Cosine Similarity Matching
- Resume Ranking
- Match Percentage Calculation
- Dynamic Job Description Input
- Data Visualization
- Export Results to CSV

---

## 🛠️ Technologies Used

- Python
- Pandas

- NLP
- TF-IDF
- Cosine Similarity

## 📂 Dataset

The dataset contains:

- Resume Category
- Job Title
- Resume Text

**Note:** The dataset is not included in this repository because of GitHub file size limits. Download the dataset separately and place it in the project folder as:

`Resume dataset.csv`

---

## 📈 Workflow

1. Load Resume Dataset
2. Explore Dataset
3. TF-IDF Vectorization
4. Convert Job Description into Vector
5. Calculate Cosine Similarity
6. Rank Resumes
7. Display Top Matching Candidates
8. Export Results

---

## 📊 Output

- Top Matching Resumes
- Match Percentage
- Best Matching Resume
